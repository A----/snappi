Snappi Test Website

Here is the website used to test snappi-selenium. It is written using Bootstrap as you may have noticed.
Snappi allows you test that your website still look the same, even when you update your libraries or custom CSS.

Using different webdrivers, you can also test that your website have a consistent look across various browsers.

You may know that Bootstrap offers convenient classes for Responsive Web Design. Try resizing your browser window. Now you can test that too!

See the table? Don't worry about Jack, he was on vacation. The ordering is done with Datatables. Testing that a table has all its entries and that they are in the correct order is quite a job to do. Well, not anymore.
