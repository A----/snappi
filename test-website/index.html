Snappi Test Website

Here is the website used to test snappi-selenium. It is written using Bootstrap as you may have noticed.
Snappi allows you test that your website still look the same, even when you update your libraries or custom CSS.

Using different webdrivers, you can also test that your website have a consistent look across various browsers.

You may know that Bootstrap offers convenient classes for Responsive Web Design. Try resizing your browser window. Now you can test that too!

See the table? Don't worry about Jack, he was on vacation.
The ordering is done with Datatables. Testing that a table has all its entries and that they are in the correct order is quite a job to do. Well, not anymore.

What about this chart or any generated image? You could do some assertions on the SVG paths, but Highcharts (the javascript library used) can also generate VML on old Internet Explorer browser. So double the job.

Testing the rendered image is easier and more meaningful and convenient.

It is also easy to put in place and maintain. A test is failing since Steve just changed an outdated component, so false positive? No need to rewrite new Selenium expectations! Check that everything looks good, and set that as reference from now on.
